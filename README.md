# Mattype

Typing Practice Application Built using React and Django (Djoser) (and SCSS)
Typing Paragraphs- Fetched from Wikipedia, (https://wikipedia.org) which uses CC BY-SA 4.0 https://creativecommons.org/licenses/by-sa/4.0/deed.en
Leaderboard- based on avg and best WPM
Stats Page- displays graph of user WPM and time
Heatmap- colors keyboard keys based on user accuracy, supports hover feature to see accurate presses / times key should have been pressed
